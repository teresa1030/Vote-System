# 簡易版活動票選系統
後端使用Java SpringBoot建立、前端使用Vue.js開發一個簡易版的活動票選系統

## 功能介紹
* 瀏覽、新增、修改、刪除活動
* 投票、瀏覽票選選項目與票數

## 開發環境
後端
  * Spring Boot
  * MySQL

前端
  * Vue.js
